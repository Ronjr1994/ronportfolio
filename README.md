# Ron Work's Portfolio

Static GitHub Pages-ready portfolio package with enhanced animations and hover effects.

## Publish to GitHub Pages

1. Upload the contents of this folder to the root of your repository.
2. Make sure `index.html` is in the repository root.
3. Keep `.nojekyll` in the root.
4. In GitHub repo settings, go to Pages.
5. Set Source to `Deploy from a branch`.
6. Set Branch to `main` and folder to `/(root)`.
