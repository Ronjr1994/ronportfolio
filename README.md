# Ron Work's Portfolio — FINAL Master Version

This is the final master ZIP package for the static GitHub Pages portfolio.

Included:
- index.html
- .nojekyll
- assets/css/styles.css
- assets/js/app.js
- assets/data/site.json
- assets/icons/
- assets/media/

Notes:
- Upload the contents of this folder to the root of your GitHub repository.
- Keep `.nojekyll` in the repo root.
- For GitHub Pages, use: Settings → Pages → Deploy from a branch → main → /(root)
- Project URL format: https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/
